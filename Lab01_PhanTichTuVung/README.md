# IT3323 Compiler Construction
### Run Project Command
- **Windows:** 
    - **With argv:** `gcc -o scanner ./*.c ; ./scanner "./test/example1.kpl"`
    - **Without argv:** `gcc -o scanner ./*.c ; ./scanner`
- **Linux:** 
    - **With argv:** `gcc -o scanner ./*.c && ./scanner "./test/example1.kpl"`
    - **Without argv:** `gcc -o scanner ./*.c && ./scanner`
